## Usage Guidelines

### Instagram Content
- **Carousel Posts**: Use 3-slide sets for educational content, drill tutorials, or tips
- **Grid Posts**: Maintain visual consistency with golden accents and dark backgrounds
- **Highlights**: Replace default icons with branded circular covers
- **Guides**: Create comprehensive educational content with branded cover

### Multi-Platform Strategy
- **LinkedIn**: Professional tone, business-focused messaging
- **Pinterest**: Educational, step-by-step content for coaches and players
- **Discord**: Community-building, engagement focus
- **Email**: Clean, branded headers for newsletters and campaigns
- **YouTube**: Consistent branding across all video content

### Marketing Campaigns
- **Social Ads**: Test different formats (square, vertical) for platform optimization
- **Display Ads**: Use compact leaderboard format for web campaigns
- **App Store**: Showcase core features with branded screenshots
- **Referral**: Encourage social sharing with clear value proposition
- **Promotions**: Create urgency with time-limited offers

---

## Technical Specifications

### File Formats
- **PNG**: Used for graphics requiring transparency or sharp text (Instagram posts, LinkedIn, Pinterest)
- **JPG**: Used for photographic elements and covers (Guides, Highlights, LinkedIn cover)

### Resolution Standards
- All assets are high-resolution, platform-optimized
- Instagram: 1080px width minimum
- LinkedIn: 1584px width for covers
- Pinterest: 1000px minimum for pins
- Display Ads: Platform-specific dimensions

### Color Management
- sRGB color space for web display
- WCAG AA compliant contrast ratios
- Consistent brand colors across all assets
