## üìê Template Editing Guidelines

### **Text Zones**
1. **Title Zones:** Use electric gold (#E8B14E), bold weight
2. **Body Text:** White/light gray for readability
3. **CTAs:** Gold gradient backgrounds with black text
4. **Captions:** Keep under 125 characters for social media

### **Image Overlays**
1. **Drill Cards:** Replace instruction area with drill demo photo/video
2. **Player Highlights:** Add player photo in background (30% opacity)
3. **App Promos:** Insert phone mockup with app screenshot

### **QR Codes**
1. Use high-contrast QR code (black on white)
2. Position in bottom-right corner (drill cards)
3. Size: 150√ó150px minimum for scannability
